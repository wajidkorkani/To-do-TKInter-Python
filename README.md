# To-do TKInter Python

A simple and user-friendly to-do list application built with Python using **CustomTkinter** for a modern graphical user interface.

## Features

- ✅ **Add Tasks** - Easily add new tasks to your to-do list
- ✅ **Mark Complete** - Mark tasks as complete when finished
- ✅ **Delete Tasks** - Remove tasks from your list
- ✅ **Task Status Tracking** - View the status of each task (incomplete/complete)
- ✅ **Modern UI** - Built with CustomTkinter for a clean, modern interface

## Requirements

- Python 3.7 or higher
- CustomTkinter

## Installation

1. Clone or download this repository:
   ```bash
   git clone <repository-url>
   cd To-do-TKInter-Python
   ```

2. Install the required dependency:
   ```bash
   pip install customtkinter
   ```

## Usage

Run the application from the `Src` directory:

```bash
python Src/app.py
```

### How to Use:

1. **Enter a Task** - Type your task in the text field
2. **Add Task** - Click the "Add Task" button to add it to your list
3. **Mark Complete** - Click "Mark Complete" to mark a task as done
4. **Delete** - Click "Delete" to remove a task from the list
